# 🎓 Student Course Finder (Python + SQLite)

## 📌 Overview
A simple project to manage students and their courses using Python and SQLite.

## 🚀 Features
- Add courses
- Add students
- Link students with courses using course_id
- View students with their courses (using JOIN)
- Search student by name

## 🧠 Concepts Used
- SQLite database
- SQL queries (INSERT, SELECT, JOIN, WHERE)
- Python loops and input handling

## ▶️ How to Run
1. Install Python
2. Run the file:
   python student_course_finder.py
